2. The Three Log Levels

Level When to Use Detail Level
OFF Production (Default) Nothing is written. Zero overhead, zero latency impact.
MIN Business Audit 2 rows per turn: user question + final answer with end-to-end duration.
VERBOSE Engineering Debug Every activity row — full chain of thought with per-step timing and raw payloads.

Log Level Guide:
• OFF — Maximum speed, no logging overhead. Recommended for production.
• MIN — Track user questions and final answers only. Good for usage analytics and auditing.
• VERBOSE — Full debug mode. Captures every tool call, LLM step, and response timing.
